Personleg trening heime

Trening som kjem til deg

Du treng ikkje kjenne deg klar for eit treningssenter. Eg møter deg heime eller ute og tilpassar treninga til kroppen, kvardagen og måla dine.

Trygt, personleg og utan unødvendig prestasjonspress.

Ein roleg start

Ein tryggare måte å kome i gang på

Når treninga skjer i kjende omgjevnader, kan terskelen bli lågare og merksemda liggje på det som faktisk betyr noko for deg.

Tilbodet passar både for deg som er heilt ny til trening, og for deg som har trena før, men har hatt ein lengre pause. Vi tek utgangspunkt i forma di, tidlegare erfaringar og det kvardagen gir rom for.

Du treng ikkje prestere eller vere i form før vi møtest. Eg hjelper deg med struktur, tydeleg rettleiing og ein fast avtale som gjer det lettare å gjennomføre.

Enkelt utgangspunkt

Du treng ikkje eit treningsrom

Vi kan kome i gang utan at du kjøper utstyr eller gjer om heimen din.

Kroppsvekt, stolar, trapper, strikk og vanleg golvplass kan gi mange gode moglegheiter. Har du utstyr frå før, kan vi bruke det når det passar inn i opplegget.

Om litt enkelt utstyr kan vere nyttig seinare, kan eg gi ei nøktern tilråding. Oppfølginga skal handle om treninga di – ikkje om å selje utstyr.
