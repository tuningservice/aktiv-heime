# Prosjektreglar for Aktiv Heime

- Alt synleg innhald skal vere på naturleg norsk nynorsk.
- Legg aldri tekst, overlegg, knappar, kort eller dekor over ansikt i bileta.
- Behald eit varmt, tillitsfullt, roleg og eksklusivt uttrykk.
- Finn aldri på kvalifikasjonar, kundehistorier, prisar, omtalar, helsepåstandar eller kontaktopplysningar.
- Ikkje legg inn sporing eller analyse utan uttrykkeleg godkjenning.
- Hald sida statisk og kompatibel med GitHub Pages.
- Test mobiloppsettet etter kvar visuelle endring.
- Kontroller alltid at alle biletreferansar fungerer.
